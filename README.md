# react-hook-form

Simple React project demonstrating form handling with react-hook-form.

## Description

Lightweight example app showing how to build accessible, performant forms using react-hook-form and modern React.

## Features

- Declarative form state management
- Built-in validation support
- Minimal re-renders
- Easy integration with UI libraries

## Prerequisites

- Node.js >= 14
- npm or yarn

## Install

```bash
# clone repository
git clone https://github.com/Fatima-Rahmani79/react-hook-form.git
cd react-hook-form

# install dependencies
npm install
# or
yarn
```

## Usage

Start the dev server:

```bash
npm start
# or
yarn start
```

## Testing

Add tests as needed. Example:

```bash
npm test
# or
yarn test
```
